# Управление столовой
Автоматизированная информационная система школьного питания. Проект реализует личные кабинеты ученика, повара и администратора, учёт оплат и выдачи питания, заявки на закупку и отчётность.
## Ключевые возможности
- регистрация и авторизация пользователей
- просмотр меню завтраков и обедов
- оформление оплаты (разово и абонемент)
- выдача питания по коду
- учёт выданных блюд
- контроль остатков продуктов
- заявки на закупку продуктов
- статистика оплат и посещаемости
- отчёт по питанию в формате XLSX
- отзывы о блюдах
## Технологии
- Python 3.10+
- Django 4.2.7
- PostgreSQL
- HTML/CSS/JS
- openpyxl

## Установка и запуск
1. Создайте и активируйте виртуальное окружение.
2. Пропишите команду `pip install -r requirements.txt`
3. Настройте подключение к PostgreSQL в `app/settings.py`.
4. Выполните миграции.
5. Запустите сервер разработки.

Пример команд:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
